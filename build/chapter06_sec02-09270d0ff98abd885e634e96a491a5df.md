# 6.2 Basic Differentiation Rules

```{admonition} Warning: under construction
:class: warning

```

Knowing the derivative of each elementary function is only the beginning. Real engineering functions are built by combining simpler ones through scaling and addition. We need rules that translate these operations on functions into corresponding operations on their derivatives. The constant rule, the factor rule, and the sum rule form the basic toolkit that makes differentiation of polynomials and similar expressions straightforward.

```{admonition} Learning goals
:class: attention
* [ ] You can apply the **constant rule**: the derivative of a constant function is zero.
* [ ] You can apply the **factor rule**: $\bigl(c \cdot f(x)\bigr)' = c \cdot f'(x)$ for any constant $c$.
* [ ] You can apply the **sum rule**: $\bigl(f(x) + g(x)\bigr)' = f'(x) + g'(x)$.
* [ ] You can differentiate any **polynomial** by combining the sum rule, the factor rule, and the power rule from the previous section.
```
