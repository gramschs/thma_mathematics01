```{admonition} Learning goals
:class: attention
* [ ] You know the definition of an **antiderivative**: a function $F$ is an
  antiderivative of $f$ on an interval if $F'(x) = f(x)$ for all $x$ in that
  interval.
* [ ] You know that if $F$ is one antiderivative of $f$, then every
  antiderivative has the form $F(x) + C$ for an arbitrary constant $C$.
* [ ] You can find antiderivatives of power functions, the exponential
  function, and the basic trigonometric functions by reversing the
  differentiation rules from Chapter 6.
```
