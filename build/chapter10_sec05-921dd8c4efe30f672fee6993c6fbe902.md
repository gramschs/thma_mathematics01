```{admonition} Learning goals
:class: attention
* [ ] You know that $G(x) = \int_a^x f(t)\,dt$ defines a new function of the
  upper limit $x$, called an **accumulation function**.
* [ ] You know the second part of the Fundamental Theorem of Calculus:
  $G'(x) = f(x)$, that is, differentiation undoes integration.
* [ ] You can interpret $G(x)$ physically as the accumulated quantity (area,
  displacement, charge) between a fixed starting point $a$ and a variable
  endpoint $x$.
```
