# The Average Value of a Function

```{admonition} Learning goals
:class: attention
* [ ] You know the definition of the **average value** of a continuous
  function $f$ on $[a, b]$: $\bar{f} = \dfrac{1}{b - a}\int_a^b f(x)\,dx$.
* [ ] You can interpret the average value geometrically as the height of a
  rectangle over $[a, b]$ whose area equals the area under the curve.
* [ ] You are familiar with the **Mean Value Theorem for Integrals**: there
  exists at least one point $c \in [a, b]$ such that $f(c) = \bar{f}$, and
  you can locate $c$ for concrete examples.
```
