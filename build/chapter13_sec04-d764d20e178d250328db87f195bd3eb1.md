# Modulus and Argument

```{admonition} Learning goals
:class: attention
* [ ] You know the definition of the **modulus** $|z| = \sqrt{a^2 + b^2}$ as
  the distance from the origin to the point $z = a + bi$ in the complex plane.
* [ ] You know the definition of the **argument** $\arg(z)$ as the angle
  between the positive real axis and the vector representing $z$, measured
  counterclockwise.
* [ ] You can compute the modulus and argument of a given complex number and
  locate it in the complex plane using these two quantities.
```
