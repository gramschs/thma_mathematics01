Composition of Functions

So far every function we have studied takes a number as its input and produces a number as its output. But what if we feed the output of one function directly into a second function as its input? This chaining of two functions into a single operation is called the composition of functions, and it is one of the most important constructions in mathematics. It is also, as we will see in a later chapter, the foundation of the chain rule in differential calculus.

Learning goals¶

Learning goals

  • You understand the concept of composition of functions and know the notation

    (f∘g)(x)=f(g(x)).(f \circ g)(x) = f(g(x)).(f∘g)(x)=f(g(x)).

  • You can compute f∘gf \circ gf∘g and g∘fg \circ fg∘f for given functions.

  • You know that composition is in general not commutative, that is, f∘g≠g∘ff \circ g \neq g \circ ff∘g=g∘f.

  • You can identify the inner function and the outer function in a composite expression.

Chaining two marble tracks¶

Imagine we connect two marble tracks in series. The ball rolls down the first track and arrives at its finishing block at a certain height. It then continues onto a second track, where this arrival height determines its starting position. The second track transforms the height it receives into a new height at its own finishing block. The entire system takes one input, the starting position on the first track, and produces one output, the final height at the end of the second track. Two functions have been chained into one.

Let us make this concrete. The first track transforms a position xxx into a height u=g(x)u = g(x)u=g(x). The second track transforms a position uuu into a height y=f(u)y = f(u)y=f(u). The combined system maps xxx directly to yyy:

y=f(u)=f(g(x)).y = f(u) = f(g(x)).y=f(u)=f(g(x)).

This combined mapping is the composition of fff and ggg.

What is ... a composition of functions?

Let g:Xg→Ygg: X_g \to Y_gg:Xg​→Yg​ and f:Xf→Yff: X_f \to Y_ff:Xf​→Yf​ be two functions such that the range of ggg is contained in the domain of fff, that is, g(Xg)⊆Yfg(X_g) \subseteq Y_fg(Xg​)⊆Yf​. The composition of fff and ggg is the function

(f∘g):Xg→Yf,x↦f(g(x)).(f \circ g): X_g \to Y_f, \quad x \mapsto f(g(x)).(f∘g):Xg​→Yf​,x↦f(g(x)).

We call ggg the inner function and fff the outer function.

The notation f∘gf \circ gf∘g is read as “fff after ggg”, which reflects the order of operations: we first apply ggg, then apply fff to the result.

Computing compositions¶

To compute a composition (f∘g)(x)=f(g(x))(f \circ g)(x) = f(g(x))(f∘g)(x)=f(g(x)), we proceed in two steps: first we evaluate the inner function ggg at xxx, then we substitute the result into the outer function fff.

Example 1: As a first example, let g(x)=x2+1g(x) = x^2 + 1g(x)=x2+1 and f(u)=uf(u) = \sqrt{u}f(u)=u​. We compute f∘gf \circ gf∘g:

(f∘g)(x)=f(g(x))=f(x2+1)=x2+1.(f \circ g)(x) = f(g(x)) = f(x^2 + 1) = \sqrt{x^2 + 1}.(f∘g)(x)=f(g(x))=f(x2+1)=x2+1​.

We substitute the entire expression x2+1x^2 + 1x2+1 in place of the argument of fff. The domain of f∘gf \circ gf∘g is all xxx for which g(x)=x2+1≥0g(x) = x^2 + 1 \geq 0g(x)=x2+1≥0, which is satisfied for all real xxx. So the domain of f∘gf \circ gf∘g is R\mathbb{R}R.

Example 2: As a second example, let g(x)=3x−1g(x) = 3x - 1g(x)=3x−1 and f(u)=u2f(u) = u^2f(u)=u2. Then:

(f∘g)(x)=f(g(x))=f(3x−1)=(3x−1)2=9x2−6x+1.(f \circ g)(x) = f(g(x)) = f(3x - 1) = (3x - 1)^2 = 9x^2 - 6x + 1.(f∘g)(x)=f(g(x))=f(3x−1)=(3x−1)2=9x2−6x+1.

Notice that a composition of two simple functions can produce a more complex expression. In the first example, fff and ggg were both straightforward, but their composition x2+1\sqrt{x^2 + 1}x2+1​ is a function we would not have written down directly. Recognising it as a composition will be essential when we differentiate it later.

A note on domains: the output values of the inner function ggg must lie within the domain of the outer function fff. If f=uf = \sqrt{u}f=u​ requires u≥0u \geq 0u≥0, then we need g(x)≥0g(x) \geq 0g(x)≥0 for all xxx in the domain of f∘gf \circ gf∘g. Checking this condition is part of computing the composition correctly.

Order matters¶

On our two marble tracks, connecting track A into track B is physically not the same as connecting track B into track A. The same is true for compositions: swapping the order of fff and ggg generally produces a different function.

Using the same functions as before, g(x)=x2+1g(x) = x^2 + 1g(x)=x2+1 and f(u)=uf(u) = \sqrt{u}f(u)=u​, we now compute g∘fg \circ fg∘f instead:

(g∘f)(x)=g(f(x))=g(x)=(x)2+1=x+1.(g \circ f)(x) = g(f(x)) = g(\sqrt{x}) = (\sqrt{x})^2 + 1 = x + 1.(g∘f)(x)=g(f(x))=g(x​)=(x​)2+1=x+1.

The domain of g∘fg \circ fg∘f is [0,∞)[0, \infty)[0,∞), since f(x)=xf(x) = \sqrt{x}f(x)=x​ requires x≥0x \geq 0x≥0.

Comparing the two results:

f∘g:x↦x2+1,D=R,f \circ g: x \mapsto \sqrt{x^2 + 1}, \quad D = \mathbb{R},f∘g:x↦x2+1​,D=R,
g∘f:x↦x+1,D=[0,∞).g \circ f: x \mapsto x + 1, \quad D = [0, \infty).g∘f:x↦x+1,D=[0,∞).

These are clearly different functions, with different expressions and different domains. Composition is not commutative: in general, f∘g≠g∘ff \circ g \neq g \circ ff∘g=g∘f.

Identifying inner and outer functions¶

When we encounter a composite expression written as a single formula, we need to be able to identify the inner and outer functions. This skill becomes essential for the chain rule: before we can differentiate a composite function, we must recognise it as one.

The key question is: what is the last operation performed? That last operation is the outer function fff, and everything inside it is the inner function ggg.

Consider h(x)=(3x2−5)4h(x) = (3x^2 - 5)^4h(x)=(3x2−5)4. The last operation is raising something to the power 4, so the outer function is f(u)=u4f(u) = u^4f(u)=u4. Everything inside the brackets is the inner function: g(x)=3x2−5g(x) = 3x^2 - 5g(x)=3x2−5. We can verify: f(g(x))=(3x2−5)4=h(x)f(g(x)) = (3x^2 - 5)^4 = h(x)f(g(x))=(3x2−5)4=h(x).

Summary and outlook¶

The composition of functions chains two functions in sequence: the output of the inner function becomes the input of the outer function. Composition is not commutative, and the order of fff and ggg must always be checked carefully. Recognising a composite expression and naming its inner and outer functions is a skill that will be needed immediately when we reach differential calculus: the chain rule states precisely how to differentiate a composition, and it requires exactly this decomposition.
