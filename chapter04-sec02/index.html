4.2 Inverse Functions

At the end of the previous section we were left with a question we could not yet answer: at what time does the capacitor reach a given voltage? We know how to compute the voltage for any time ttt, but now we need to run that process in reverse. This idea of undoing a function is one of the most important constructions in mathematics, and it leads us to the concept of the inverse function.

Learning goals¶

Learning goals

  • You know what an inverse function is and understand the notation f−1f^{-1}f−1.

  • You can check whether a given function is invertible.

  • You know that any strictly monotonic function is invertible.

  • You can calculate the inverse function by solving the function equation for xxx.

  • You know that the graph of f−1f^{-1}f−1 is the mirror image of the graph of fff about the line y=xy = xy=x.

  • You can apply the identities f(f−1(y))=yf(f^{-1}(y)) = yf(f−1(y))=y and f−1(f(x))=xf^{-1}(f(x)) = xf−1(f(x))=x.

Reversing the capacitor question¶

The discharging curve from the previous section tells us the voltage across the capacitor at any given time:

UC(t)=U0⋅e−t/τ.U_C(t) = U_0 \cdot e^{-t/\tau}.UC​(t)=U0​⋅e−t/τ.

Suppose we ask: after how much time has the voltage dropped to half its initial value? We need to find the time ttt for which UC(t)=0.5⋅U0U_C(t) = 0.5 \cdot U_0UC​(t)=0.5⋅U0​. Substituting into the formula:

U0⋅e−t/τ=0.5⋅U0.U_0 \cdot e^{-t/\tau} = 0.5 \cdot U_0.U0​⋅e−t/τ=0.5⋅U0​.

Dividing both sides by U0U_0U0​:

e−t/τ=0.5.e^{-t/\tau} = 0.5.e−t/τ=0.5.

We need to find the value of −t/τ-t/\tau−t/τ for which the exponential equals 0.5. In other words, we need to undo the exponential function. This is the problem of finding the inverse function.

The general situation is the following. We have a function fff that maps an input xxx to an output y=f(x)y = f(x)y=f(x). Given yyy, we want to find xxx. If we can do this uniquely, that is, if every output value yyy came from exactly one input value xxx, then we can define a new function that maps yyy back to xxx. This new function is the inverse of fff.

When is a function invertible?¶

Not every function can be inverted. Consider the squaring function f(x)=x2f(x) = x^2f(x)=x2. If we are told that f(x)=9f(x) = 9f(x)=9, we cannot determine xxx uniquely: both x=3x = 3x=3 and x=−3x = -3x=−3 are valid answers. The squaring function maps two different inputs to the same output, so it cannot be inverted over its full domain R\mathbb{R}R.

A function is invertible if and only if every output value is produced by exactly one input value. Graphically, this means that every horizontal line intersects the graph of the function at most once. This is called the horizontal line test.

What is ... an invertible function?

A function f:X→Yf: X \to Yf:X→Y is called invertible over its domain XXX if for every value yyy in the image f(X)f(X)f(X) there exists exactly one x∈Xx \in Xx∈X with f(x)=yf(x) = yf(x)=y.

The inverse function f−1:f(X)→Xf^{-1}: f(X) \to Xf−1:f(X)→X is then defined by

f−1(y)=xif and only iff(x)=y.f^{-1}(y) = x \quad \text{if and only if} \quad f(x) = y.f−1(y)=xif and only iff(x)=y.

The most useful criterion for invertibility follows directly from the properties we studied in chapter 3: any strictly monotonic function is invertible. If fff is strictly increasing, then larger inputs always give larger outputs, so no two inputs can produce the same output. The same argument applies to strictly decreasing functions.

Since the exponential function f(x)=exf(x) = e^xf(x)=ex is strictly monotonically increasing, it is invertible. This guarantees that our capacitor question has a unique answer.

Calculating the inverse function¶

To find the inverse function of a given fff, we follow a systematic procedure. Starting from y=f(x)y = f(x)y=f(x), we solve for xxx in terms of yyy. The resulting expression gives us x=f−1(y)x = f^{-1}(y)x=f−1(y). We then conventionally swap the roles of xxx and yyy so that the inverse function is also written with xxx as the input variable.

Example 1: As a first example, let us find the inverse of the linear function f(x)=3x−5f(x) = 3x - 5f(x)=3x−5:

  1. Write y=3x−5y = 3x - 5y=3x−5.

  2. Solve for xxx: add 5 to both sides to get y+5=3xy + 5 = 3xy+5=3x, then divide by 3 to get x=y+53x = \frac{y + 5}{3}x=3y+5​.

  3. Swap xxx and yyy: y=x+53y = \frac{x + 5}{3}y=3x+5​.

  4. Write as the inverse function: f−1(x)=x+53f^{-1}(x) = \frac{x + 5}{3}f−1(x)=3x+5​.

We can verify this using the two fundamental identities of inverse functions. Applying fff and then f−1f^{-1}f−1 should return the original input:

f−1(f(x))=f−1(3x−5)=(3x−5)+53=3x3=x.✓f^{-1}(f(x)) = f^{-1}(3x - 5) = \frac{(3x - 5) + 5}{3} = \frac{3x}{3} = x. \checkmarkf−1(f(x))=f−1(3x−5)=3(3x−5)+5​=33x​=x.✓

Applying f−1f^{-1}f−1 first and then fff should also return the original input:

f(f−1(x))=f ⁣(x+53)=3⋅x+53−5=x+5−5=x.✓f(f^{-1}(x)) = f\!\left(\frac{x+5}{3}\right) = 3 \cdot \frac{x+5}{3} - 5 = x + 5 - 5 = x. \checkmarkf(f−1(x))=f(3x+5​)=3⋅3x+5​−5=x+5−5=x.✓

Example 2: As a second example, let us find the inverse of f(x)=x2f(x) = x^2f(x)=x2 on the restricted domain D=[0,∞)D = [0, \infty)D=[0,∞). On this domain fff is strictly monotonically increasing and therefore invertible:

  1. Write y=x2y = x^2y=x2 with x≥0x \geq 0x≥0.

  2. Solve for xxx: x=yx = \sqrt{y}x=y​ (we take the positive root since x≥0x \geq 0x≥0).

  3. Swap xxx and yyy: y=xy = \sqrt{x}y=x​.

  4. Write as the inverse function: f−1(x)=xf^{-1}(x) = \sqrt{x}f−1(x)=x​, with domain [0,∞)[0, \infty)[0,∞).

This example illustrates an important point: when a function is not invertible over its full domain, we can often make it invertible by restricting the domain to an interval on which it is strictly monotonic. We will use exactly this idea when we define the inverse functions of sine and cosine in section 4.

Graphical interpretation¶

There is a beautiful geometric relationship between the graph of a function and the graph of its inverse. To obtain the graph of f−1f^{-1}f−1 from the graph of fff, we reflect the graph of fff about the line y=xy = xy=x. This is because reflecting about y=xy = xy=x swaps the coordinates of every point: a point (a,b)(a, b)(a,b) on the graph of fff (meaning f(a)=bf(a) = bf(a)=b) becomes the point (b,a)(b, a)(b,a) on the graph of f−1f^{-1}f−1 (meaning f−1(b)=af^{-1}(b) = af−1(b)=a).

This reflection also explains what happens to the domain and range. If f:X→Yf: X \to Yf:X→Y has image f(X)f(X)f(X), then f−1f^{-1}f−1 maps f(X)f(X)f(X) back to XXX. The domain and range are swapped. For our linear example, f(x)=3x−5f(x) = 3x - 5f(x)=3x−5 maps R\mathbb{R}R to R\mathbb{R}R, and so does f−1f^{-1}f−1. For the restricted squaring function, f:[0,∞)→[0,∞)f: [0, \infty) \to [0, \infty)f:[0,∞)→[0,∞) and f−1=xf^{-1} = \sqrt{x}f−1=x​ also maps [0,∞)[0, \infty)[0,∞) to [0,∞)[0, \infty)[0,∞).

Summary and outlook¶

A function is invertible when every output value comes from exactly one input value. Strictly monotonic functions always satisfy this condition. The inverse function is found by solving the function equation for xxx, and its graph is the reflection of the original graph about the line y=xy = xy=x. The two identities f(f−1(y))=yf(f^{-1}(y)) = yf(f−1(y))=y and f−1(f(x))=xf^{-1}(f(x)) = xf−1(f(x))=x confirm that the two functions undo each other. We are now ready to answer the capacitor question that opened this section: the inverse of the natural exponential function exe^xex is the function we call the natural logarithm, which is the subject of the next section.
