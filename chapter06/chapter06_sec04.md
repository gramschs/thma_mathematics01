# The Chain Rule

In chapter 3 we saw that composite functions appear throughout engineering: the voltage across a capacitor is an exponential function of a quantity that is itself a linear function of time. To differentiate such expressions we need a rule that handles composition directly. The chain rule is probably the most frequently applied differentiation rule in practice, and recognizing a composite expression is the first step in using it.

```{admonition} Learning goals
:class: attention
* [ ] You know the **chain rule**: if $h(x) = f(g(x))$, then $h'(x) = f'(g(x)) \cdot g'(x)$.
* [ ] You can identify the **inner function** $g$ and the **outer function** $f$ in a composite expression, as practiced in chapter 3.
* [ ] You can apply the chain rule to differentiate composite functions involving exponential, logarithmic, trigonometric, and power functions.
```
