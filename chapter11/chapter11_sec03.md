```{admonition} Learning goals
:class: attention
* [ ] You know the **integration by parts** formula
  $\int u\,dv = uv - \int v\,du$ as the integration counterpart of the
  product rule from Section 6.3.
* [ ] You can select $u$ and $dv$ strategically using the LIATE priority
  order: logarithms, inverse trigonometric functions, algebraic functions,
  trigonometric functions, exponential functions.
* [ ] You can apply integration by parts repeatedly when a single application
  does not resolve the integral.
```
