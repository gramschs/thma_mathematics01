```{admonition} Learning goals
:class: attention
* [ ] You can decompose a proper rational function into **partial fractions**
  for the cases of distinct real poles and repeated real poles.
* [ ] You know when a rational function is proper, and can reduce an improper
  rational function to a polynomial plus a proper part by polynomial long
  division before applying partial fractions.
* [ ] You can integrate each partial fraction term using the standard
  antiderivatives $\int \frac{1}{x - a}\,dx = \ln|x - a| + C$ and
  $\int \frac{1}{(x-a)^n}\,dx$ for $n \geq 2$.
```
