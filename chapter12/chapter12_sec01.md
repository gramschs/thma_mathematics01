```{admonition} Learning goals
:class: attention
* [ ] You can set up the definite integral for the **area between two curves**
  $f$ and $g$ on an interval $[a, b]$ as $\int_a^b |f(x) - g(x)|\,dx$.
* [ ] You can identify which function is the upper curve on a given interval
  and split the integral at intersection points where the curves cross.
* [ ] You can compute the area between two curves by finding the intersection
  points, determining the sign of $f(x) - g(x)$ on each subinterval, and
  evaluating the resulting integrals using the techniques of Chapter 11.
```
